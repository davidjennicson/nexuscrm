# Nexus CRM

**From your first customer to your thousandth. One platform. Every scale.**

A modern, open-source CRM platform built with React and Spring Boot. Designed for businesses scaling from scrappy startup to enterprise operations — without complexity or vendor lock-in.

---

## 🎯 The Problem We're Solving

Most businesses face this choice:
- **Simple tools** → outgrown in 6 months
- **Enterprise CRM** → overkill, expensive, needs consultants
- **Custom build** → wastes time, burns cash

**There's no middle ground.** Until now.

---

## 💡 Our Vision

**Nexus CRM is built for the journey.** 

From a solo founder tracking first sales to a multi-team organization managing complex pipelines. Same system. Your pace. Your rules.

- **Low-code interface** — no developer needed to get started
- **Pluggable architecture** — add features as you grow, not before
- **Open source** — complete transparency, your data always yours
- **Built to scale** — Spring Boot backend + React frontend, production-ready from day one

---

## ✨ Current Features

- **Contact Management** — Track all customer interactions in one place
- **Company Profiles** — Organize and manage company relationships
- **Deal Pipeline** — Visualize sales with Kanban, tables, and graphs
- **Task Management** — Create, assign, and track tasks with priorities
- **Team Collaboration** — Manage team members and roles seamlessly
- **Workflow Automation** — Automate repetitive business processes
- **Analytics Dashboard** — Real-time insights into your business metrics
- **Secure Authentication** — Protected routes, user onboarding, session management
- **Responsive Design** — Works perfectly on desktop, tablet, and mobile
- **Dark Mode** — Theme switching for comfortable workflow

## 🚀 Tech Stack

**Frontend:**
- React 19 + TypeScript
- Vite (fast builds)
- Tailwind CSS + shadcn/ui (beautiful, consistent UI)
- React Router, React Query, Recharts
- Animations with Motion

**Backend:**
- Spring Boot + Spring Security
- PostgreSQL (scalable data)
- RESTful API architecture

## 🛠️ Development

```bash
# Clone and install
git clone <repository-url>
cd nexus-crm
npm install

# Start development server
npm run dev
```

Available at `http://localhost:5173`

---

## 🏗️ Build & Deploy

```bash
npm run build      # Production build
npm run preview    # Preview production build locally
npm run lint       # Code quality check
npm run typecheck  # TypeScript validation
```
